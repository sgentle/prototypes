Quicksort (in-place)

  1. Pick a pivot number
  2. Repeatedly
    1. Find the next item from the left that is greater than the pivot
    2. Find the next item from the right that is less than the pivot
    3. Swap them
    4. Stop when there are no more items to swap
  3. Repeat this with the left half of the array
  4. Repeat this with the right half of the array
